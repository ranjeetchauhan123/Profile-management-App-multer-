# 📌 Profile Management App

A simple Node.js + Express + MongoDB based CRUD application where users can create, update, view, and reset profiles with image upload support.

---

## 🚀 Features

- ✅ Add new user profile
- ✅ Upload profile image (using Multer)
- ✅ View all profiles
- ✅ Update existing profile
- ✅ Reset profile data (without deleting document)
- ✅ Server-side validation (Mongoose Schema)
- ✅ Clean UI with form handling

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Templating Engine:** EJS  
- **File Upload:** Multer  
- **Styling:** HTML + CSS
